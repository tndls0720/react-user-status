# ☕ Cafe Menu Service

React + TypeScript로 구현한 간단한 **카페 메뉴 판매 웹 애플리케이션**입니다.  
메뉴 아이템을 클릭하여 장바구니에 추가하고, 주문 내역을 확인할 수 있는 기능을 제공합니다.

## 🚀 주요 기능

- 카페 메뉴 목록 표시
- 메뉴 항목별 가격 표시
- 주문하기 버튼 → 장바구니에 추가
- 장바구니 내역 실시간 반영
- 컴포넌트 기반 아키텍처

## 🛠️ 사용 기술

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- CSS (모듈형 스타일 가능)
- React Hooks (`useState`, `props` 전달 등)
